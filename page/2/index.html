Window Thin PC 安装注册组件配置

发表于 2012-09-23   |   分类于 Win   |  


10/5更新状况-今天突然激活过期了，桌面显示黑屏，提示今天必须注册。
比较奇怪，之前安装时候已经用证书注册过了，用slmgr.vbs -dli和-dlv 发现出现两个系统信息版本VOLUME MAK 和 VOLUME BA。VOLUME MAK 显示过期了。
想到很可能是之前在装了一些embedded的系统组件(.Net Framework)造成的吧。
网上找了下，发现一个写好的批处理工具(仍然使用了之前的证书，但是多使用了一个文件和系列号)重新注册了下，重启就Ok了。

Windows 7 瘦身版Windows Thin PC（简称WinTPC），WinTPC是一个面向企业用户的产品，主要面向虚拟桌面基础架构（VDI）消费者，WinTPC由于组件的精简，消耗系统资源更少，对于普通用户，如果你的电脑配置较低，对系统功能的要求也不高，WinTPC将是一个很好的选择，特别合适仅需要虚拟机安装个讯雷会员下载什么的。WinTPC下载（目前只有英文版,但可以完美汉化）及有效激活方法和一些必要组件的安装：

下载安装 MNDN版下载：
ed2k://|file|en_windows_thin_pc_x86_697681.iso|1576980480|2D0E6A048EB3F314F556B4F0834A95E2|h=HST4XIJCM7BXSDWD6OXDBBONACMB7I76|/
评估版（90天免费使用）下载：http://download.microsoft.com/download/C/D/7/CD789C98-6C1A-43D6-87E9-F7FDE3806950/ThinPC_110415_EVAL_x86fre.iso
有效激活方法，使用证书文件激活:
1,下载证书文件，给个地址但是可能已经无效(点击此处下载)
2,将证书文件放到文件位置 C:\Windows\System32\spp\tokens\skus\Security-SPP-Component-SKU-Embedded 中
3,然后管理员权限运行 slmgr.vbs –rilc 即可成功激活!

汉化方法

中文语言包下载：http://download.microsoft.com/download/8/6/1/8616D57C-1163-45FB-832A-15FA60571002/SP1/Chinese%20(Simplified)%20Language%20Pack%20-%20Windows%20Embedded%20Standard%207%20SP1/lp.cab%20Language%20Pack%20-%20Windows%20Embedded%20Standard%207%20SP1/lp.cab)
Windows Thin PC 简体中文包的安装方法：
1.下载Windows Thin PC 简体中文语言包lp.cab放到D盘的根目录
2.以管理员身份运行CMD命令输入：dism /online /add-package /packagepath:D:lp.cab
3.完成后输入：bcdedit /set {current} locale zh-cn
4.完成后再输入：bcdboot %WinDir% /l zh-cn
完成后重启计算机进入控制面板进行设置即可。

组件安装(以需要安装Windows Live Writer为例)

因为WinTPC精简了很多组件，所以安装一些软件时会提示缺少一些系统组件，这时我们需要让系统再补上它们，但是千万不要什么都补，否则就直接装Win 7 标准版了，不要折腾WinTPC了.

以Windows Live Writer为例安装.Net Framework 3.0, 3.5：
1.下载需要的组件winemb-netfx30.cab与winemb-netfx35.cab放到D盘的根目录。
下载Windows Embedded Standard 7 SP1 Runtime (IBW) x86 ISO ，可以到官方下载评估版本http://www.microsoft.com/download/en/details.aspx?id=11887。功能扩展包组件位于ISO的 DSPackagesFeaturePack 文件夹下, 从中可找到winemb-netfx30.cab与winemb-netfx35.cab。
2.以管理员身份运行CMD命令输入：dism /online /add-package /packagepath:D:**.cab
3.完成后重启计算机
4.可成功安装Windowns Live Writer

好了，如果以后WinTPC安装什么软件或者一些功能需要时就可以通过类似方法补上了。
